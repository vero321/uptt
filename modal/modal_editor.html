{$prueba}

Para los párrafos se le debe dejar una linea en blanco para dividirlos entre si.

Para los subtitulos se le coloca al lado izquierdo el sigo igual ¨=¨ para identificarlos. Ejm:
=subtitulo
Para las listas se le debe colocar al lado izquierdo un asterisco ¨*¨ con espacio. Ejm:
* Gato
* Perro
* Pollo
Para las listas enumeradas se le debe colocar al lado izquierdo un numeral ¨#¨ con espacio. Ejm:
# Gato
# Perro
# Pollo
Para las citas textuales se le debe colocar el signo de mayor ">" y luego la cita. Ejm:
>Este es el texto indicando una cita textual cualquuiera
Para las imágenes se debe definir dentro de corchetes dobles ¨[[]]¨ el tipo, el nombre y la descripción. Ejm:
[[img img_iteración Iteración 01]]
